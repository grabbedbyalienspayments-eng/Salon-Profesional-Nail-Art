
# Shine & Shape Nails Studio

Website one-page pentru salon de manichiură și pedichiură premium în București.

## Deployment pe Vercel

Acest proiect este pregătit pentru deployment pe Vercel.

### Pași pentru deployment:

1. Creează un cont pe [Vercel](https://vercel.com)
2. Conectează repository-ul GitHub
3. Vercel va detecta automat configurația Vite
4. Click pe "Deploy"

### Configurare manuală (dacă este necesar):

- **Framework Preset**: Vite
- **Build Command**: `npm run build`
- **Output Directory**: `dist`
- **Install Command**: `npm install`

## Dezvoltare locală

```bash
# Instalare dependențe
npm install

# Pornire server de dezvoltare
npm run dev

# Build pentru producție
npm run build

# Preview build
npm run preview
```
